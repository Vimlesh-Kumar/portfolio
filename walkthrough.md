# Portfolio Walkthrough

This project is a modern, single-page React application built with **Vite**, **Framer Motion**, and **Lucide React**. It's designed to showcase your skills and experience as a Full Stack Developer with a premium, high-end feel.

## Key Features

- **Modern UI/UX**: Dark-themed design with glassmorphism effects and custom purple/blue gradients.
- **Dynamic Animations**: Smooth scroll-reveal animations and interactive hover states.
- **Responsive Design**: Fully optimized for desktop and mobile devices.
- **Profile Avatar**: A professional digital illustration was generated to represent you as a modern "active developer".

## Project Structure

- `src/App.jsx`: Contains the entire application logic and layout.
- `src/index.css`: Defines the global design system, color tokens, and utility classes.
- `src/assets/profile.png`: Your professional avatar.

## How to Run Locally

1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`

## Deployment Instructions (Free Hosting)

I recommend using **Vercel** or **Netlify** for free hosting. Both offer seamless integration with GitHub.

### Step 1: Push to GitHub

1. Create a new repository on [GitHub](https://github.com/new).
2. Run the following commands in your terminal:
   ```bash
   git init
   git add .
   git commit -m "Initial portfolio commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```

### Step 2: Deploy to Vercel

1. Go to [Vercel](https://vercel.com/new) and log in with GitHub.
2. Select your repository.
3. Click **Deploy**. Vercel will automatically detect Vite and host your site.

### Step 3: Deploy to Netlify (Alternative)

1. Go to [Netlify](https://app.netlify.com/start) and log in with GitHub.
2. Select your repository.
3. Click **Deploy Site**.
